Analysis classes.

twpasolver.analysis.analysis_function(func)[source]#

Wrap functions for analysis.

Automatically saves results of each analysis function in data dictionary of an Analyzer class. All analysis functions must return a dictionary to be compatible with this wrapper.

phase_matching(thin: int = 20) → dict[str, Any][source]#

Build phase matching profile.

The phase matching condition is computed considering all signal frequencies lower than the first stopband and all pump frequencies from the end of the first stopband.

Parameters:

thin (int) – The step size to thin out the frequency and wavenumber arrays.

Returns:

A dictionary containing:
